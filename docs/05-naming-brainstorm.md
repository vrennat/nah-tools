# Naming Brainstorm

## Naming Criteria

The name needs to:
1. **Be short** — ideally 1-2 syllables, definitely under 12 characters
2. **Signal "free" or "open"** — without sounding cheap
3. **Be memorable** — mentioned once, you remember it
4. **Have an available .com** (or compelling alternative TLD)
5. **Not sound like the scammy competitors** — no "QR Code Generator Pro Plus Premium"
6. **Work as a brand** — could extend to other tools later
7. **Be globally pronounceable**

The name should NOT:
- Include "free" (sounds desperate/suspicious)
- Be generic enough to confuse with existing tools
- Require explanation
- Sound corporate

---

## The Competitive Naming Landscape

**Predatory / subscription tools (the enemy):**
- QR Code Generator / QR Code Generator Pro (Bitly)
- QR Tiger (qrcode-tiger.com)
- QR.io
- Flowcode (flowcode.com)
- Uniqode (uniqode.com)
- QRfy, QR Creator, QR Code Creator

**Free / freemium tools:**
- QR Code Monkey (secretly same company as QR Code Generator Pro)
- GoQR (goqr.me)
- ME-QR (me-qr.com)

**Open-source projects:**
- Mini QR (mini-qr.com) — cute, small scope
- qrframe — HN hit, technical focus
- Reactive Resume (rxresu.me) — good naming precedent

**Naming patterns in the space:**
- "QR + [noun]": QR Tiger, QR Monkey — overused, blends together
- "[prefix]QR": MeQR, GoQR — slightly better, still generic
- "QR + [made-up word]": Uniqode, Flowcode — enterprise-y, VC feel
- Single clean words: nobody has claimed one

## Decision Framework

**Question 1: Standalone or suite?**
- Standalone → QR-specific name (PlainQR, BlipQR, NullQR)
- Suite → Brand name (Openly, Zero Tools) with QR as first product

**Question 2: Primary audience?**
- Small business owners → approachable (Plain, Stamp, Ink)
- Developers → clever/technical (Null, Glyph, Scanline)
- Both → short and neutral (Blip, Speck)

**Question 3: How aggressive is the messaging?**
- Subtle → name implies quality (Plain, Clear, Openly)
- Moderate → name implies freedom (Uncaged, Libre)
- Full send → calls out the problem directly (Not a Scam QR)
